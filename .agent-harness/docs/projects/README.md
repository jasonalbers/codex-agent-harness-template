# Projects

Project-specific harness notes generated by `node .agent-harness/dist/cli.js project bootstrap` live here.

This directory intentionally contains no real project notes in the public
template. Copy `.agent-harness/config/projects.example.json` to `.agent-harness/config/projects.json`, then run
`node .agent-harness/dist/cli.js project bootstrap` for private or project-specific setup.

Tracked example docs may be committed when they describe known projects. Generated local drafts for private projects should be reviewed before committing.
