# Connect GitHub Repo

1. Confirm `gh auth status`.
2. Confirm repo access:

```bash
gh repo view owner/repo
```

3. Add repo metadata to `.agent-harness/config/projects.json`.
4. Confirm branch naming in `.agent-harness/docs/github/branching.md`.
5. Confirm PR expectations in `.agent-harness/docs/github/pull-request-model.md`.
