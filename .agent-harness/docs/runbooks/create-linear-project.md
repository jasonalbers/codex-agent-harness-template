# Create Linear Project

1. Open the shared Linear workspace.
2. Create a project named after the GitHub repo.
3. Configure statuses from `.agent-harness/docs/linear/linear-status-model.md`.
4. Add labels from `.agent-harness/docs/linear/linear-labels.md`.
5. Copy the project slug from the URL.
6. Add the slug to `.agent-harness/config/projects.json`.
