# Agent Operating Model

Humans steer. Agents execute.

Linear supplies the work. GitHub stores code. This base repository stores the
operating system for how agents work.

Agents must:

- Start from a Linear issue.
- Read `AGENTS.md` and `.agent-harness/WORKFLOW.md`.
- Plan before editing.
- Use isolated workspaces.
- Keep changes small.
- Run verification.
- Produce proof of work.
- Open a GitHub PR.
- Stop for human review when required.
