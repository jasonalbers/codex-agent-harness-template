# QUALITY_SCORE.md

## Quality Score

Use this rubric before completing meaningful work.

Score each area from 0 to 3:

- 0: missing or misleading.
- 1: partial and fragile.
- 2: usable with known gaps.
- 3: clear, verified, and maintainable.

## Areas

- Product clarity.
- UI/UX clarity for any user-facing work.
- Agent legibility.
- Architecture boundary.
- Verification.
- Security and data safety.
- Reliability.
- Documentation freshness.
- Future maintainability.

## Completion Rule

Do not claim work is complete if any required area scores 0. Record gaps in the
plan or tech debt tracker.
