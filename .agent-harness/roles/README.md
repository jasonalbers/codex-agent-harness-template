# Role Instructions

This directory contains role-specific instructions for AI tools that interact with this repository.

Root `AGENTS.md` is intentionally short and acts as the router.

## Roles

- `chatgpt-web.md` - ChatGPT web with GitHub connector. Used for idea intake, product refinement, and Idea Pack creation.
- `codex.md` - Codex implementation worker. Used for scoped code or documentation changes.
- `shared-rules.md` - Safety, quality, and source-of-truth rules shared by all roles.

## Operating Principle

ChatGPT web thinks and refines.

Codex implements reviewed work.

Linear tracks build work.

GitHub stores code, docs, branches, pull requests, and proof of work.
